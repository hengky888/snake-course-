# Python Snake Game

A simple, classic Snake game built with Python and `pygame`.

## Requirements

- Python 3.x
- `pygame` library

## Installation

1. Navigate to the `snake_game` directory.
2. Install the required library using pip:

   ```bash
   pip install -r requirements.txt
   ```

## How to Play

Run the game:

```bash
python main.py
```

- Use the **Arrow Keys** to move the snake.
- Eat the **green food** to grow and increase your score.
- Avoid hitting the walls or yourself!
- If you lose:
  - Press **C** to play again.
  - Press **Q** to quit.
